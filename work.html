Movie API

The objective of this project was to build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies. This the is firts part of the full MERN stack application called MyFlix, and was created using MongoDB, Express.js, Node.js, Mongoose, Heroku, NPM.

Meet App

This serverless, progressive web application (PWA) is built with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events. It's key features let users filter events by city, show/hide event's details and specify the number of events to be displayed. This app can be used offline, users can add a shortcut to their screen and see a chart of number of upcoming events by city.
